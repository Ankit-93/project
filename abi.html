Automated Benefit Integration (ABI)

Project Overview: AI-Driven Workflow Automation for Sales and Benefit Configuration

Objective

Automated Benefit Integration (ABI) is an AI-powered solution designed to streamline and automate workflows from sales enablement to benefit configuration. It serves as a digital front door for Sales and Account Management teams, enabling them to:

  • Accurately capture client modifications and plan adjustments.
  • Eliminate manual paperwork and reduce human errors.
  • Improve implementation efficiency and accelerate deployment.
  • Enhance automation for seamless integration with downstream systems.

Example Change Text: "Update air ambulance coins to $45 INN"

Solution Approach

1. Data Preprocessing

  • Data Collection & Cleaning: Extracted and processed "change text" from the FOBS dataset.
  • Text Normalization: Tokenized text, removed special characters, standardized numerical values, and lowercased the text.
  • Handling Class Imbalance: Applied Down Sampling to ensure fair representation across different categories.

2. Intent Prediction using ELECTRA

  • Model Choice: Fine-tuned ELECTRA (Efficiently Learning an Encoder that Classifies Token Replacements Accurately) for high-accuracy classification.
  • Architecture:
    • Input: Tokenized "change text".
    • Output: Intent classification into categories such as Modify, Add, Remove, Update, Clarify, etc.
    • Loss Function: Categorical Cross-Entropy for multi-class classification.
    • Fine-tuning: Applied transfer learning on domain-specific data.

3. Entity Extraction using RoBERTa

  • Model Choice: Fine-tuned RoBERTa for Named Entity Recognition (NER).
  • Entities Extracted:
    • Benefit Name (e.g., Air Ambulance, Urgent Care)
    • Benefit Option (e.g., Deductible, Copayment)
    • UOM (e.g., "$45", "10%", "per visit")
    • Network (e.g., "In-Network", "Out-of-Network")

4. Evaluation & Optimization

  • Evaluation Metrics:
    • Intent Prediction: 90% F1-score
    • Entity Extraction: 85% accuracy
  • Optimization Strategies:
    • Hyperparameter tuning using Bayesian Optimization.
    • Handled class imbalance via re-weighting loss functions.
    • Explored ensemble learning (BERT + RoBERTa) for robustness.

Challenges & Solutions

Challenge Solution
Ambiguous Change Texts Fine-tuned ELECTRA with domain-specific data to improve intent understanding.
Complex Entity Structures Used RoBERTa + CRF for overlapping entities and ensured correct classification of related fields.
Pipeline Integration Built a modular AI pipeline for intent prediction, classification, and entity extraction.

Impact & Results

  • Accuracy Gains:
    • Intent Prediction: ~90% accuracy
    • Entity Extraction: ~85% accuracy
  • Efficiency Improvements: 40% reduction in processing time.
  • Enhanced Data Insights: Extracted granular data points for improved downstream analytics.
  • Scalability & Integration: Designed for seamless integration into existing FOBS workflows.

Conclusion

Automated Benefit Integration (ABI) leverages AI-driven intent prediction, classification, and entity extraction to revolutionize benefit configuration. By automating manual processes, improving accuracy, and accelerating decision-making, ABI enhances operational efficiency and scalability for sales and account management teams.
